# Raycasting engine in C and SDL2
Engine like Wolfenstein 3D, renders false 3D scenes using tilemaps and raycasting.

## Build and run instructions
1. **Generate build files and get dependencies:**
    ```bash
    cmake -B build
    ```
2. **Compile and run:**
    ```bash
    cmake --build build --target run
    ```

## Controls
* **Mouse:** Look around
* **W/S:** Forwards/backwards
* **A/D:** Left/right
* **Esc:** Exit the programme
